# SaaS Landing Page Conversion Audit

Audit a SaaS landing page to identify conversion blockers and sales positioning issues that prevent $10K+ enterprise deals.

## Audit Checklist

### 1. Hero Section (First 3 Seconds)
- [ ] **Clear value proposition** — Can a visitor understand what it does in 5 seconds?
- [ ] **Specific outcome** — Does it promise a measurable result ("32% more jobs") or just features?
- [ ] **Primary CTA** — Is there ONE dominant action? ("Request demo" beats "Learn more")
- [ ] **Social proof** — Logos, numbers, or testimonials above the fold?

### 2. Pricing Page (Critical for $10K+ Sales)
- [ ] **Pricing is visible** — No pricing = enterprise buyers assume it's too cheap or hidden
- [ ] **Tiered options** — Starter / Professional / Enterprise tiers
- [ ] **Enterprise tier** — Must exist even if it says "Contact us" — signals you handle big deals
- [ ] **ROI framing** — Price should be anchored to value ("$2,497/month to capture $50K in storm jobs")
- [ ] **No "Free" as primary** — Free trials are fine, but don't lead with free for B2B SaaS

### 3. Credibility Signals
- [ ] **Live stats work** — Counter animations showing real numbers (not "0+" or broken counters)
- [ ] **Specific testimonials** — Named person, company, city, specific result (not generic praise)
- [ ] **Case studies** — Before/after with numbers: "Premier Roofing went from $200K to $400K monthly"
- [ ] **Compliance badges** — SOC 2, GDPR, etc. (only if actually true)
- [ ] **Demo video** — "Watch it work" must actually play a video, not just be text

### 4. Product Demo Section
- [ ] **Interactive or video demo** — Screenshots alone don't sell software
- [ ] **Shows the full workflow** — Not just one screen, but the end-to-end process
- [ ] **Labeled screenshots** — Each image needs a caption explaining what the viewer is seeing

### 5. Comparison Section (Without vs. With)
- [ ] **Specific pain points** — "Manually check weather sites" not "old way is slow"
- [ ] **Quantified losses** — "Miss 60% of storm damage leads" not "miss opportunities"
- [ ] **Toggle or side-by-side** — Let visitors switch between old way and new way

### 6. Missing Elements That Block $10K+ Sales
- [ ] **ROI calculator** — Let prospects input their numbers and see projected return
- [ ] **Implementation timeline** — "Go live in 48 hours" reduces perceived risk
- [ ] **Territory/availability** — "Limited to 3 contractors per county" creates scarcity
- [ ] **Guarantee** — "3X ROI in 90 days or we work free" removes risk
- [ ] **Live chat** — Enterprise buyers want to ask questions immediately

## Common Issues Found

| Issue | Impact | Fix |
|-------|--------|-----|
| Stats show "0" | Kills credibility | Fix JavaScript counters or use static numbers |
| No pricing page | Buyers assume it's cheap | Add 3 tiers, even if Enterprise is "Contact us" |
| Missing demo video | Can't visualize value | Record 2-min walkthrough or use Loom |
| Generic testimonials | No trust | Get specific: name, company, city, result |
| No case studies | Can't prove ROI | Create 3 detailed before/after stories |
| No comparison table | Hard to justify switch | Build old vs. new checklist |

## Output Format

Deliver audit as:
1. **What's Working** (positive findings)
2. **Critical Issues** (blocking sales — fix first)
3. **Quick Wins** (can fix in 1 day)
4. **Strategic Improvements** (need 1-2 weeks)
5. **Pricing Recommendations** (if applicable)

## Positioning for $10K+ Sales

For enterprise deals, the page must sell:
- **Outcome, not features** — "Capture every storm job in your county" not "AI storm detection"
- **Risk reversal** — Guarantee, implementation support, dedicated account manager
- **Exclusivity** — "Only 3 spots per territory" or "By application only"
- **Social proof from peers** — Testimonials from companies the prospect recognizes
- **ROI proof** — Calculator, case studies, or "average client adds $X/month"
