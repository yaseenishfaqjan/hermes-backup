# GlobalShield Roofing OS — Landing Page Audit

**URL:** https://globalshield.storeshouses.com/
**Date:** 2025-06-12
**Auditor:** Hermes AI

## What's Working

- Strong headline: "From Storm Alert to Payment Collected in One Platform" — clear outcome
- Good subheadline: "Detect hail. Auto-call homeowners. Book inspections. Dispatch crews. File insurance claims. Collect payments. All connected. All automated."
- Live activity feed creates urgency and shows real workflow
- Demo dashboard screenshot shows actual product interface
- Client logos from Georgia roofing companies add local credibility
- "Powered by Scalaro" branding suggests enterprise-grade backend
- SOC 2 Compliant badge

## Critical Issues (Blocking $10K+ Sales)

### 1. Stats Show "0" — Kills Credibility
- Section shows: "0+ Roofing Contractors", "0% More Storm Jobs", "0 hrs Saved Per Week", "0% AI Estimate Accuracy"
- JavaScript counters are broken or not connected to real data
- **Fix:** Use static numbers if dynamic counters fail: "500+ Contractors", "32% More Jobs", "9 hrs Saved/Week", "98% Accuracy"

### 2. No Pricing Page
- Clicked "Pricing" link — no pricing displayed
- For enterprise buyers, missing pricing means either:
  - Product is too cheap to mention (bad)
  - Pricing is hidden because it's embarrassing (worse)
- **Fix:** Add 3 tiers:
  - Starter: $997/month (1 territory, basic features)
  - Professional: $2,497/month (3 territories, AI voice, full features)
  - Enterprise: $4,997/month (unlimited, custom integrations, dedicated support)
  - Add "Custom Implementation: $10,000+" for done-for-you setup

### 3. Missing Demo Video
- "Watch it work" link exists but no actual video
- Screenshots are good but video converts 3x better
- **Fix:** Record 2-minute Loom walkthrough of storm detection → AI call → booking → dispatch → payment

### 4. No ROI Calculator
- Roofing companies want to know: "If I pay $2,497/month, how much more will I make?"
- **Fix:** Simple calculator: input "average job value" × "jobs per month" × "32% increase" = projected revenue

## Summary

The product looks solid. The page design is professional. But it's missing the **sales infrastructure** that justifies $10K+:
- Pricing (must exist)
- ROI proof (calculator or case studies)
- Risk reversal (guarantee)
- Social proof (specific testimonials, not just logos)
- Urgency (territory scarcity)

Fix these 5 things and the page will convert enterprise buyers.
